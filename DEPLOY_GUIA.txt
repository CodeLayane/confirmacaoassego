============================================================

=== PASSO 1: CRIAR SUBDOMÍNIO ===

1. Acesse cPanel da HostGator
2. Vá em "Subdomínios" (ou "Subdomains")
3. Crie:
   - Subdomínio: confirmacao
   - Domínio: assego.com.br
   - Pasta: public_html/confirmacaoassego
4. Aguarde propagar (pode levar até 30min)

=== PASSO 2: CRIAR BANCO DE DADOS ===

1. No cPanel, vá em "Bancos de Dados MySQL"
2. Criar novo banco:
   - Nome: confirmacaoassego
   (HostGator vai transformar em: wwasse_confirmacaoassego)

3. Criar usuário do banco:
   - Nome: confir  (ou outro que preferir)
   - Senha: CRIE UMA SENHA FORTE e anote
   (HostGator vai transformar em: wwasse_confir)

4. Vincular usuário ao banco:
   - Selecione o banco wwasse_confirmacaoassego
   - Selecione o usuário wwasse_confir
   - Marque "TODOS OS PRIVILÉGIOS"
   - Clique em "Fazer alterações"

=== PASSO 3: IMPORTAR BANCO ===

1. No cPanel, vá em "phpMyAdmin"
2. Selecione o banco wwasse_confirmacaoassego
3. Clique em "Importar"
4. Selecione o arquivo: database.sql
5. Clique em "Executar"

Depois execute também o migrate.sql:
6. Clique em "Importar" novamente
7. Selecione: migrate.sql
8. Executar

=== PASSO 4: CONFIGURAR SENHA DO BANCO ===

ANTES de subir os arquivos, edite o config.php:
- Abra config.php num editor de texto
- Na linha: define('DB_PASS', '');
- Coloque a senha que você criou no passo 2
- Exemplo: define('DB_PASS', 'suaSenhaAqui123');
- Salve o arquivo

Se o nome do banco/usuário ficou diferente, ajuste também:
- define('DB_NAME', 'wwasse_confirmacaoassego');
- define('DB_USER', 'wwasse_confir');

=== PASSO 5: SUBIR ARQUIVOS ===

1. No cPanel, vá em "Gerenciador de Arquivos"
2. Navegue até: public_html/confirmacaoassego/
3. Suba TODOS os arquivos (upload zip e extrair é mais rápido):
   
   Opção A - Upload ZIP:
   - Clique "Upload" 
   - Suba o arquivo assego-deploy.zip
   - Após upload, clique direito no zip → "Extrair"
   - Mova os arquivos da pasta extraída para a raiz de confirmacaoassego/

   Opção B - FTP (FileZilla):
   - Host: ftp.assego.com.br (ou seu IP)
   - User: seu usuário FTP
   - Pasta remota: /public_html/confirmacaoassego/
   - Suba todos os arquivos

4. Estrutura final deve ficar:

1. No Gerenciador de Arquivos, dentro de confirmacaoassego/
2. Crie a pasta: uploads
3. Clique direito → Permissões → 755

=== PROBLEMAS COMUNS ===

Erro 500:
- Verifique se o .htaccess está correto
- Verifique a versão PHP (precisa ser 7.4+, ideal 8.x)
- No cPanel → "Selecionar versão PHP" → escolha PHP 8.1 ou 8.2

Erro de conexão com banco:
- Verifique senha no config.php
- Verifique se o usuário está vinculado ao banco
- Verifique nome do banco/usuário (HostGator adiciona prefixo)

Página em branco:
- No cPanel → "Selecionar versão PHP" → Extensões
- Habilite: pdo_mysql, mysqli, mbstring, json
